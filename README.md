# Azure-koneoppiminen-ruokah-vikin-torjunnassa
Sisältö: Opinnäytetyössä tutkitaan, miten Azure Machine Learning -alustaa voidaan hyödyntää ruokahävikin vähentämiseksi vähittäiskaupassa. Työn teoreettinen osuus käsittelee ruokahävikin nykytilaa, sen aiheuttamia taloudellisia ja ympäristövaikutuksia sekä koneoppimisen roolia kysynnän ennustamisessa. Teknologian osuudessa perehdytään Azure Machine Learningin toiminnallisuuksiin, kuten AutoML-, datan esikäsittely- ja mallin julkaisun ominaisuuksiin. 

Tavoite: Työn tavoitteena on luoda proof-of-concept -ratkaisu, joka hyödyntää Azure Machine Learning -alustaa tuoretuotteiden kysynnän ennustamisessa. Ennustamisen avulla voidaan vähentää tilanteita, joissa tuotteita tilataan liikaa ja ne päätyvät hävikkiin. Työssä tutkitaan myös, miten pilvipohjainen ratkaisu tukee kestävää kehitystä, kustannustehokkuutta ja skaalautuvuutta. 

Työn lopputuloksena syntyy: -Dokumentaatio ruokahävikin ongelmasta, koneoppimisen hyödyntämismahdollisuuksista ja Azure Machine Learning -alustan soveltuvuudesta. -Käytännön toteutus (proof-of-concept): pieni datamalli, jonka avulla luodaan Azure ML -ennustemalli kysynnän ennustamiseen. 

-Analyysi tuloksista: mallin tarkkuus, sen rajoitteet, sekä arvio siitä, miten ratkaisu voisi käytännössä vähentää hävikkiä vähittäiskaupassa. -Opas mallin hyödyntämiseen ja sen jatkokehitysmahdollisuuksiin.
